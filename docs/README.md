# Household Income and Expenditure Dashboard

## Overview
This project presents a household analytics dashboard focused on **income, taxation, poverty classification, and expenditure patterns** across multiple countries, with a detailed expenditure drill-down for Estonia.

The dashboard was designed to transform household-level data into clear, decision-oriented insights through interactive KPI cards, charts, and locality-based comparisons.

## Key Highlights
- Total income across the dashboard scope: **1.2B**
- Number of households analyzed: **41.7K**
- Average income: **27.6K**
- Estonia drill-down:
  - Total income: **9.2M**
  - Total expenditure: **8.2M**
  - Record count: **2.8K**

## Analytical Focus
The dashboard explores:
- country-level income distribution
- households with no employment and no unemployment help
- tax participation patterns
- poverty classification
- income-to-expenditure ratio
- expenditure breakdown by category
- locality-based differences in health and food expense percentages

## Insights
- Household income and tax participation vary substantially across countries.
- The poverty classification view shows a clear difference between households above and below the poverty line.
- Estonia’s expenditure profile suggests housing is the dominant cost category.
- Food expense percentages are relatively balanced across capital, other city, and rural localities.
